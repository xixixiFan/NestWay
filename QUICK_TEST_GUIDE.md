## 🔧 常见问题排查

### 问题 1：应用无法启动

**可能原因**：
- Supabase URL 配置错误
- 网络连接问题

**解决方案**：
1. 检查 [supabase_service.dart](file:///e:/ai/solotrip/NestWay/lib/services/supabase_service.dart) 中的 URL
2. 确保网络连接正常

---

### 问题 2：SOS 历史记录为空

**可能原因**：
- RLS 策略阻止了数据读取
- 数据库中没有测试数据

**解决方案**：
1. 在 Supabase SQL Editor 中执行：
```sql
SELECT * FROM sos_logs;
```
2. 如果查询为空，重新插入测试数据：
```sql
INSERT INTO sos_logs (id, user_id, type, location_description, triggered_at)
VALUES 
  (1, 1, 'call', '东京涩谷站附近', '2025-05-01T22:10:00Z'),
  (2, 1, 'sms', '新宿歌舞伎町', '2025-05-02T23:45:00Z'),
  (3, 1, 'video', '池袋西口', '2025-05-03T21:30:00Z');
```

---

### 问题 3：无法发送短信

**可能原因**：
- 权限问题
- 数据库写入失败

**解决方案**：
1. 检查 RLS 策略是否允许 INSERT
2. 在 SQL Editor 中手动测试插入：
```sql
INSERT INTO sos_logs (type, location_description)
VALUES ('test', '测试插入');
```
