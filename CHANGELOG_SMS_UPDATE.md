# SOS短信内容更新日志

## 📅 更新时间
2026-05-20

## 🎯 更新目标
优化SOS求助短信的文案，使其更加友好和人性化。

## 📝 变更内容

### 旧版短信内容
```
【NestWay】紧急求助！[用户]发起了SOS求助，您是TA的紧急联系人。
当前位置：[地址]（[坐标]）。请立即确认TA的安全！
```

### 新版短信内容
```
[用户昵称]向你发送了TA的实时位置，TA可能需要你的帮助！
请及时与TA联系并关注TA的动态行踪。
当前位置：[地址]（[坐标]）
(你是TA的紧急联系人，因此收到了此信息)
```

## 🔧 技术变更

## 🚀 部署步骤

### 步骤1：更新阿里云短信模板
```
1. 登录阿里云短信控制台
2. 创建新模板或修改现有模板
3. 模板内容：${name}向你发送了TA的实时位置，TA可能需要你的帮助！请及时与TA联系并关注TA的动态行踪。当前位置：${location}(你是TA的紧急联系人，因此收到了此信息)
4. 等待审核通过
5. 记录新的模板CODE
```

### 步骤2：更新Supabase配置（如需要）
```bash
supabase secrets set ALIBABA_SOS_TEMPLATE_CODE=新的模板CODE
```

### 步骤3：部署Edge Function
```bash
supabase functions deploy send-sos-sms
```

### 步骤4：测试验证
```
1. 打开App测试短信发送
2. 确认短信内容格式正确
3. 验证位置信息准确
```
